Plongez dans Python 3

Plongez dans Python 3 couvre Python 3 et ses différences avec Python 2. En comparaison avec Plongez dans Python, 20% du contenu a été révisé et 80% contient des nouveautés. Le livre est maintenant terminé, mais les retours sont toujours bienvenus.
